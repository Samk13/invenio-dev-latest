InvenioRDM GitHub Archiver (IGA) är ett verktyg utvecklat av California Institute of Technology. Det automatiserar processen att arkivera GitHub-programvaruutgåvor till en InvenioRDM-baserad lagringsserver. Verktyget kan köras manuellt från kommandoraden eller automatiskt med hjälp av GitHub Actions.

IGA är särskilt användbart för att synkronisera repository-status mellan GitHub och InvenioRDM. Det extraherar metadata från GitHub-utgåvor och repositories, konstruerar en metadata-post i det format som krävs av InvenioRDM, och skickar posten tillsammans med källarkivet för GitHub-utgåvan till InvenioRDM-servern
